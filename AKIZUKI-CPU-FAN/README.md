# 秋月 Raspberry Pi 用ユニバーサル基板と25mm角ファン 

RPi3 は 高負荷で長時間連続稼動させると、
CPU温度及び周辺温度が高温になると思います。

秋月電子より発売されている RPi用ユニバーサル基板を
ドリルで多少改造し、5V 25mm 角 DC ファンを取り付けるだけで、
結構放熱されるようです。

CPUにヒートシンクが取り付けられている
場合は干渉する可能性があります。
